Configuración de Visual Studio Code para Java

Descarga de Visual Studio

Configuración

Una vez tenemos instalado VSCode instalamos la extensión de java. En las extensiones buscamos java y seleccionamos Extension Pack for Java


Y seguimos los pasos de la instalación

Descargamos e instalamos la versión 17 de jdk




Y finalmente le damos como finalizado y ya podremos empezar. Más información: VSCode tutorial

Primera clase java

  1. Pulsamos Abrir carpeta y seleccionamos una nueva carpeta (crearemos una nueva) en la que dejaremos nuestros archivos java, por ejemplo: ejemplosjava
  2. Sobre el nombre de carpeta a la derecha marcamos el + para crear un nuevo archivo

  3. Escribimos class en la parte del código y nos completará con la plantilla de la clase

  4. Escribimos main y completará con el método

  5. Escribimos Sysout y nos completará el código y pondremos un mensaje genérico.
  6. Pulsamos ejecutar y se mostrará el mensaje.

Primer proyecto java

Crearemos un nuevo proyecto con nuestras iniciales, llamado tema1minombreyapellidos (cada uno el suyo), en mi caso tema1jagd.

Pasos:
  1. Abrir Visual Studio Code
  2. Menú Ver-> Paleta de comandos->Java: create java Project->No build tools
  3. Elegimos la carpeta destino de nuestro ordenador
  4. Ponemos el nombre de nuestro proyecto (temaxxx)
  5. Si me pregunta si confío en los autores indico sí.

Vemos que crea la estructura de carpetas y un archivo de prueba

Configuración GIT con VSCode

Requisitos previos:
  1. Descargar e instalar git de la página:
  2. Una cuenta de GitHub
  3. Tener conocimientos básicos de los conceptos de Git, como los repositorios y la clonación, como se describe en Uso de las herramientas de control de versiones de Git en Visual Studio Code.
  4. instalar GitHub Pull Request and Issues extensión. En el propio VSCode pulsamos el icono de las extensiones y tecleamos github.
  5. Una vez instalada la extensión
  6. Ingresamos usuario/contraseña y ya podemos clonar el repositorio.
  7. Le damos a Ver, Paleta de comandos (Ctrl+Shift+P) y nos bajamos el siguiente repositorio: https://github.com/spring-guides/gs-spring-boot.git

Origen de la palabra informática

Según la Real Academia Española de la Lengua, la palabra informática pasa al español a través del francés informatique, contracción de las palabras information, que traduce ‘información’, y automatique, ‘automática’, aunque su origen se registra en el alemán informatik.
